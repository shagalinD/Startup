#1. Блочный удлиннитель
**Проблема:** Пользователи часто сталкиваются с неудобством при использовании стандартных удлинителей: либо не хватает определенных разъемов (например, USB или Type-C), либо, наоборот, присутствуют лишние, которые не используются и занимают место.
**Идея:** Разработка и производство модульной системы уделителей «собери сам». Покупатель приобретает базовый блок (кабель + питание) и отдельные блоки-модули (с евророзетками, портами USB-A, USB Type-C). Пользователь самостоятельно соединяет нужное количество и тип модулей, получая персонализированный удлинитель под свои текущие задачи.
**Финансы:** Разработка дизайна и инженерии, производство самой модели, размещение на маркетплейсах
**Конкуренты:** На рынке существуют модульные решения, но они либо находятся в премиум-сегменте (дорогие дизайнерские варианты), либо предлагают ограниченную модульность.

#2. Поиск приложений
**Проблема:** Среднестатистический пользователь смартфона устанавливает десятки приложений, со временем иконки теряются на рабочих столах, и вспомнить точное название нужного приложения бывает сложно. Пользователь тратит время на пролистывание меню.
**Идея:** Создать систему быстрого поиска по приложениям на устройстве по ключевым словам (например: еда -> приложения супермаркетов и доставок)
**Финансы:** Финансы складываются из среднего показателя зарплат 4 программистов, что примерно равляется 280 тыс.р.
**Конкуренты:** В данный момент конкурентов в этой сфере нет, а строенный посик на устроиствах работает ограниченно.

#3. Система сортировки приложений
**Проблема:** Сейчас у людей на девайсах установлено огромное количество приложений, из-за чего некоторые приложения могут теряться и забываться, мы уберем эту проблемы и поможем структурировать и группировать приложения
**Идея:** Система, которая автоматически сортирует приложения по тематическим папкам
**Финансы:** Финансы складываются из среднего показателя зарплат 4 программистов, что примерно равляется 280 тыс.р.
**Конкуренты:** В данный момент конкурентов в этой сфере нет.
